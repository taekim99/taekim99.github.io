CoreCRM


CoreCRM is a powerful Customer Relationship Management (CRM) system built with Django. It helps businesses manage their customer data, streamline communication, and improve overall efficiency. This open-source project provides a user-friendly interface for recording and managing customer information.


  • User Authentication and Registration: CoreCRM ensures a secure environment by implementing robust user authentication and registration processes.

  • Customer Record Management: Seamlessly add, update, and delete customer records, providing a dynamic platform for maintaining accurate and up-to-date information.

  • Responsive and Intuitive Interface: With a focus on user experience, using Bootstrap, CoreCRM boasts a responsive and intuitive interface, making navigation and interaction seamless.

  • Customizable Forms: Tailor CoreCRM to your specific needs with customizable forms, allowing you to capture customer details in a way that aligns perfectly with your business requirements.

  • Secure Authentication with Django: Leveraging the built-in security features of Django, CoreCRM ensures that user authentication remains airtight and resilient against potential threats.

  • Data Base Management: CoreCRM harnesses the robust capabilities of MySQL for efficient and secure database management, enhancing data storage, retrieval, and overall system performance.

Sortify


The Sortify is a Python script designed to automate the organization of files within a specified directory. The main goal is to categorize files into different folders based on their types, providing a systematic approach to managing and accessing your files. Whether you have documents, presentations, music, images, videos, or other file types, the File Organizer ensures a structured and tidy file system.


  • Automatic Categorization: The script automatically categorizes files based on their extensions into predefined folders, such as documents, presentations, music, images, and videos.

  • Customizable File Types: Customize the script to recognize and organize specific file types according to your needs. Easily extend the list of supported file extensions.

  • Directory Handling: The script handles not only individual files but also entire directories. If it encounters a directory, it organizes the directory itself into a specified "unknown" folder or another appropriate category.

  • Avoids Duplicates: The script checks for existing files in the destination folders to avoid duplicates. If a file already exists, it skips the move operation and notifies you.
